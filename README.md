**Descripción del proyecto**
Este proyecto se desarrollo en el Sprinti 5, y tiene como objetivo analizar datos relacionados con la economía de movilidad, con el fin de identificar patrones, tendencias y comportamientos relevantes en el uso de servicios.
A través de técnicas de limpieza de datos, análisis exploratorio y cálculos estadísticos, se busca transformar datos crudos en información útil para la toma de decisiones.

Este repositorio incluye:

(S5_ladb_mobility_economy_project_student.ipynb)

Cálculo de métricas
**Datasets**
Archivos utilizados para el análisis (incluidos o cargados desde ruta).
**Resultados y hallazgos**
Insights obtenidos a partir del análisis de los datos.

**Objetivo del análisis**
Entender el comportamiento de los datos de movilidad
Identificar patrones relevantes
Generar insights accionables
